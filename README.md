# Calculator

A simple calculator built with HTML, CSS, and JavaScript as part of The Odin Project Foundations curriculum.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Clear (AC) button
* Supports multi-digit numbers
* Supports chained operations (e.g. `2 + 3 + 4`)

## What I Learned

This project helped me practice:

* DOM manipulation
* Event listeners
* JavaScript functions
* State management
* Handling user input
* Building interactive web applications

## Future Improvements

* Add decimal support
* Add keyboard support
* Improve calculator styling
